前言

操作

安装chrome

说明：由于某些众所周知原因，我们在正常的网上冲浪的时候不能访问谷歌商店，自然就不能从官方网站那里直接安装谷歌浏览器了。但是，我们可以从腾讯软件商店等（老马打钱）或者热心网友的分享来获得我们所需要的软件。

既然我们作为伟大社会主义国家，那么就必须给这个浏览器加上一层社会主义的色彩它才能在这里混下去

设置chrome

设置->地址栏中的搜索引擎->百度/搜狗/360

到这里，它已经是一个合格的社会主义接班人社会主义浏览器了。

那么它属于资产阶级的一面要怎么办呢？当然是师夷长技以制夷啦

谷歌上网助手

下载插件：Ghelper解压

在浏览器地址栏输入

1、勾选开发者模式 2、加载已解压的扩展程序

找到解压的路径选择ghelper_source文件夹，仔细阅读”源码安装方法.txt“

免费注册并登录,新用户会获得3天的免费vip，登录之后就可以上Google了

Setup VPN

从Chrome网上应用商店安装SetupVPN

注册，登录，这个插件目前声称永久免费(FREE Forever)

登录之后建议选择日本节点就可以上油管了，速度还不错：
