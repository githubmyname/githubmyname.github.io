hexo博客的搭建-笔记

本博客主要根据win教程搭建而成win+hexo部署则参考学长的教程Hexo博客搭建Linux+hexo

一、环境搭建

1、git

2、Node.js

3、hexo

二、本地建站

1、例如新建文件夹F:\blog\xxx.github.io，并在此目录下右键gitbash进入命令行界面

2、安装hexo：npm install -g hexo-cli

3、淘宝镜像：npm config set registry “https://registry.npm.taobao.org"

4、初始化：hexo init

npm install

5、本地预览：hexo server(或者hexo s),在浏览器地址栏输入localhost:4000访问博客默认页面

markdown写法请参考：官方文档

三、部署到github/Coding上

1、github/Coding：代码托管仓库，可以免费提供公开域名以便分享自己的博客

2、建议使用Coding，因为它的服务器位于国内，一般来说无论访问速度还是上传速度都优于github

3、 安装插件：npm install hexo-deployer-git –save

注册/登录coding，新建项目，blog

修改_config.yml

修改coding地址：

![](2018-08-20-about\coding.png)

4、保存，hexo deploy

5、开启coding的静态page服务，部署来源选择master分支，保存后自动运行，通过访问地址给他人分享你的博客
